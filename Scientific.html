How to work with a scientific calculator

Calculator keys to perform trigonometric functions

[ sin ] -Returns the sin of x (x is in radians) ;
[ cos ] - Returns the cos of x (x is in radians);
[ tan ] - Returns the tan of x (x is in radians);
[ π ] - is a mathematical constant, the ratio of a circle’s circumference to its diameter;
[ asin ] - Returns the arccosine of x, in radians;
[ acos ] -Returns the arccosine of x, in radianse;
[ atan ] - Returns the arccosine of x, in radians;
[ e ] - is a mathematical constant, Euler’s number;
[ log ] - is the common logarithm (to the base 10);
[ ln ] - is the natural logarithm (to the base e);
[ % ] - return remainder of the two numbers after division;
[ 1/x ] - return inverse of a number;
[ √ ] - calculates the square root;
[ ∛ ] - calculates the square root;
[ x^2] - raises X to the power of 2;
[ x^3] - raises X to the power of 2;
[ 00] - is the key for entering 2 zeros;
[ n!] - return factorial of number;
[ e^x] - raises e to the power of x;
[ 10^x] - raises 10 to the power of x.

Functions of the standard buttons

[ 0 ], [ 1 ], [ 2 ], ... [ 9 ] - standard number keys;
[ ←] - remove the last character on the display;
[ + ] - addition, [ - ] - subtraction, [ х ] - multiplication, [ ÷ ] - division;
[ C ] - resets the calculator .
